# NEXUS - Conectando dados, transformando trajetórias
> Plataforma para gestão e integração de dados voltados à prevenção da evasão escolar.  
> Projeto desenvolvido no segundo semestre da SPTech. 🚧 Em desenvolvimento.

---

## 📌 Sobre o Projeto

O **NEXUS** é uma plataforma desenvolvida com o objetivo de apoiar instituições de ensino superior privadas no monitoramento e análise de dados relacionados à evasão escolar.

Através da centralização e integração de informações, o sistema busca auxiliar na identificação precoce de alunos em risco, contribuindo para ações preventivas e estratégias de permanência escolar.

---

## 🎯 Objetivo

- Monitorar indicadores acadêmicos  
- Integrar dados sobre desempenho e frequência  
- Gerar insights para apoio à tomada de decisão  
- Contribuir para a redução da evasão escolar  

---

## 🏫 Problema

A evasão escolar é um desafio significativo no contexto educacional brasileiro, impactando diretamente o desenvolvimento social e profissional dos estudantes.

A falta de integração de dados dificulta a identificação antecipada de padrões que podem indicar risco de abandono.
